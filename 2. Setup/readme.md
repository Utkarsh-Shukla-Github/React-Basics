#   Setup

-   Download & Install `NodeJs`.
-   Then `create react app`. 

There are two ways to **create-react-app**

| `npx`      | `npm` |
| ----------- | ----------- |
| `npm create-react-app <project_name>` | `npm install create-react-app -g`|
| *npm package runner*   | `create-react-app <project_name>`|
